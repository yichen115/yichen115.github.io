找调试口

2022-03-05

找 GND

使用万用表，调到通断档，黑表笔接到电路上，红表笔依次测试，如果响了就表示是 GND
使用万用表，调到电压档，黑表笔接到电路上，红表笔依次测试，如果电压为 0 表示是 GND

找 TX

如果串口在不断的输出数据，TX 引脚的电压是不断变化的，所以黑表笔接 VCC，红表笔试一下，电压不断变化的就是 TX

找 RX

RX 引脚默认高电平，与 VCC 的电压差为 0

其实 RX 与 TX 来回多试试吧，也不一定准吧

奇奇怪怪的调试口

耳机

把有线耳机拆开，大致能分出几条线，用万用表通断档测试一下具体是哪根线连着哪一部分，带着麦克风的耳机线是有四部分，比如小米的一个耳机（开始带货：https://m.tb.cn/h.fsKQy9t?tk=Iurl2QA06e3）
